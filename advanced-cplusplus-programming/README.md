## AI Use Policy

### ✅ Allowed

- Asking AI to **explain a concept** (e.g., “What is a loop?”)
- Requesting **simple examples** to study and learn from
- Using AI to **clarify theory or terminology**

### ❌ Not Allowed

- Asking AI to **complete or solve assignment tasks** for you
- **Copying and pasting** AI‑generated code directly into your submissions

### Why This Matters

By skipping the **building phase**, you miss the essential learning that develops true programming skill.
If you don't write the code yourself, you'll struggle later to:

- Understand and extend existing codebases
- Identify and fix errors in AI‑generated code
- Build the analytical mindset that defines a real developer

In the future, developers will likely **validate and improve AI‑generated code** more than they'll write it from scratch.
But validation requires deep expertise, expertise you can only gain by **building things manually now**.

### Recommended Use

If you choose to use AI, prefer study modes, for examle [**ChatGPT Study Mode**](https://openai.com/index/chatgpt-study-mode/) - but **use it mindfully and ethically**.

### ⚠️ Privacy Notice

ChatGPT is **not an official university tool**. It's provided by an external company (OpenAI), and any data you share is processed outside the university's systems.

**Never share:**

- Student ID or any personal identifiers
- Email, phone number, or address
- Login credentials or passwords
- Confidential project, company, or research information
- Sensitive personal data about yourself or others

By using ChatGPT, you acknowledge that **you are responsible for any information you provide**.
