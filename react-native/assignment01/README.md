# Assignment 01 - React Native intro, Expo setup, first screen

**Estimated time:** ~10h

## Required Video Notes

- `notes/video01.pdf`

## What You Should Learn

- What React Native is
- How Expo projects are created and run
- How to use `View`, `Text`, `Image`, and `Pressable`
- How state changes what is shown on screen

## Mini Project

Build a **Student Profile Card App**.

## Points Checklist

- ✅ Expo project runs successfully: 2 p
- ✅ Profile image is shown in the app: 1 p
- ✅ Name, degree programme, and short bio are shown: 2 p
- ✅ At least 4 skills are shown: 1 p
- ✅ One reusable information component is used: 2 p
- ✅ One interactive button works: 2 p
- ✅ Button causes a visible state change: 2 p
- ✅ Basic styling and spacing are clear: 2 p

**Total: 14 p**
