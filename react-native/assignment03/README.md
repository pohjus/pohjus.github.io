# Assignment 03 - Forms, controlled inputs, validation

**Estimated time:** ~11h

## Required Video Notes

- `notes/video01.pdf`
- `notes/video02.pdf`

## What You Should Learn

- How controlled inputs work
- How to validate user input
- How to show useful validation feedback
- How to manage form state cleanly

## Mini Project

Build a **Student Wellness Check-in App**.

## Points Checklist

- ✅ Form has at least 5 fields: 2 p
- ✅ At least 3 different input types are used (e.g., `TextInput` with
  different `keyboardType` values, `Switch`,
  `@react-native-picker/picker`): 2 p
- ✅ Required field validation is implemented: 2 p
- ✅ Numeric or length-based validation is implemented: 2 p
- ✅ Error messages are shown to the user: 2 p
- ✅ Successful submission shows a summary view: 2 p
- ✅ Reset button works: 1 p

**Total: 13 p**

Example fields:

- name
- email
- study programme
- stress level
- sleep hours
- free text comment
