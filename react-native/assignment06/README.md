# Assignment 06 - Images, media picking, ScrollView, polished UI

**Estimated time:** ~13h

## Required Video Notes

- `notes/video01.pdf`
- `notes/video02.pdf`

## What You Should Learn

- How to use images in React Native
- When to use `ScrollView`
- How to let the user pick media
- How to polish a mobile layout

## Mini Project

Build a **Travel Journal App Screen**.

## Points Checklist

- ✅ Destination header section exists: 1 p
- ✅ At least 3 content sections inside a `ScrollView`: 2 p
- ✅ Gallery or memory section exists: 2 p
- ✅ Call-to-action (CTA) button exists: 1 p
- ✅ User can select at least one image with Expo Image Picker: 3 p
- ✅ Layout has polished spacing and section structure: 2 p
- ✅ Visual styling is consistent and readable: 2 p

**Total: 13 p**
