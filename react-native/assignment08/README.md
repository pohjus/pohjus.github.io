# Assignment 08 - Persistence, context, theming, accessibility

**Estimated time:** ~14h

## Required Video Notes

- `notes/video01.pdf`
- `notes/video02.pdf`

## What You Should Learn

- How persisted state differs from temporary state
- How to use context for shared state
- How to support light and dark themes
- How to improve accessibility basics

## Mini Project

Build a **Notes App** or **Budget Tracker App**.

## Points Checklist

- ✅ Add item works: 2 p
- ✅ Edit item works: 2 p
- ✅ Delete item works: 2 p
- ✅ Data persists between app restarts: 3 p
- ✅ Shared state uses context: 2 p
- ✅ Light/dark theme toggle works: 2 p
- ✅ Basic accessibility improvements are included: 2 p
- ✅ App remains readable and usable in both themes: 2 p

**Total: 17 p**

Minimum accessibility requirements:

- accessible button text
- clear labels for important inputs
- enough contrast between text and background
