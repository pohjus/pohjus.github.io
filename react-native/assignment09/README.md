# Assignment 09 - Device features, permissions, location, media

**Estimated time:** ~16h

## Required Video Notes

- `notes/video01.pdf`
- `notes/video02.pdf`

## What You Should Learn

- How device permissions work
- How to read the current location
- How to combine media and device data
- How to design a feature flow around platform capabilities

## Mini Project

Build a **Location Journal App**.

## Points Checklist

- ✅ App asks for location permission: 2 p
- ✅ Current coordinates are read successfully: 3 p
- ✅ Journal entry can be saved with location: 3 p
- ✅ Image can be attached from the device: 3 p
- ✅ Saved entries are shown in a list (in-memory storage is
  sufficient): 2 p
- ✅ Detail view for one entry exists: 2 p
- ✅ Permission denied state is handled gracefully: 3 p

**Total: 18 p**
